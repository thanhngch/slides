Javascipt

Ngôn ngữ phổ biến nhất năm 2018 (thống kê của Stackoverflow)

Được dùng chủ yếu trên trình duyệt và server (NodeJS)

Đã được chuẩn hóa ECMA

Phiên bản phổ biến ES5 và ES6

Variable

Primitive và object

Primitive gồm: undefined, number, boolean, string, Symbol

Còn lại là object; array, function cũng là object

Variable (tiếp)

number, boolean, string tự động chuyển sang kiểu Object nếu thấy cần thiết

number -> Number, boolean -> Boolean, string -> String

Ví dụ (1).toString()

Ngôn ngữ js là ngôn ngữ kiểu động (không cần khai báo kiểu cho biến) và yếu (biến có thể thay đổi kiểu và dữ liệu bất cứ lúc nào)

Number

Chỉ có dạng số thực không có kiểu nguyên

Ngoài ra còn có NaN, -Infinity, Infinity

Số 0 có hai dạng hai số 0 và -0

Biểu diễn số lớn sử dụng e: 1e3 == 1000

Sử dụng toán tử +, -, *, / và %

Sử dụng ** để tính lũy thừa (ES6)

Number (tiếp)

Sử dụng thư viện Number và Math

Chuyển dữ liệu về kiểu Number

Number.parseInt(), Number.parseFloat()

Toán tử + ví dụ +"10" // 10

Number (tiếp)

Lưu ý 0.1 + 0.2 != 0.3

và 9999999999999999 == 10000000000000000

Sử dụng Number.isSafeInteger()

String

Gồm nhiều character UTF-16 (1 hoặc 2 code unit)

Hàm: length, substring, replace, search, split, toUpperCase, indexOf, charAt

"" và '' có gì khác biệt

Sử dụng `` trong ES6

Phân biệt substr, substring

String

substr(from, length) và substring(from, to)

Chuyển dữ liệu về String: a.toString()

Boolean

true hoặc false

Dữ liệu nào chuyển về là kiểu false

false, 0, string rỗng (“), NaN, null và undefined; sẽ trở thành false

Còn lại là true

Boolean (tiếp)

Chuyển về kiểu Boolean

Boolean(a), !(a)

Toán tử &&, || và !

Truthy và Falsy

Boolean (tiếp)

[] là truthy nhưng nó == false

Ví dụ if ([]) {...}

if (a) {} sẽ thực thi khi a là truthy

!!a sẽ tương đương với Boolean(a)

Boolean (tiếp)

Các toán tử && liên tiếp

var name = o && o.name()

Các toán tử || liên tiếp

var name = getName() || 'bill'

Khai báo biến

var, let(ES6), const (ES6)

var là function scope

Hosting: khai báo var ở đâu trong hàm cũng cho lên đầu hàm

let, const là block scope

const không phải là hằng mà nó sẽ không trỏ được đến giá trị khác

Control flow

for, while, do while, if, if else, switch case, ternary( ? : )

Object

Dữ liệu dạng key : value, trong đó key là string và value là tùy ý

Khai báo tương tự JSON nên JSON dùng rất tốt trong JS

obj[‘name’] và obj[name] là khác nhau

var a = {
    [name]: 'bill'
}
var a = {}; a[name] = ‘bill’ // same above

Array

Là một object đặt biệt với các key là số được sắp xếp theo thứ tự

Các hàm sử dụng : concat(), join(), pop(), push(), shift(), unshift(), slice(), split(), splice()

Dùng forEach (trong ES5) thay cho for (i = 0, i < n; i++) {}

Array (tiếp)

Kiểm tra Array rỗng

if (a == []) {...} (cách sai)

Array.isArray(a) và a.length == 0

Function

Funtional: return function, variable function, parameter is funtion

Hosting và Closure

IIFEs (Immediately Invoked Function Expressions)

(function() {
    // nội dung hàm
})();

What is this

let user = {
    name: "John",
    hi() { console.log(this.name); }
}
user.hi(); // John
var hi = user.hi;
hi(); // error because this is undefied

Custom Object

Toán tử new và con trỏ this

function Person(first, last) {
    this.first = first;
    this.last = last;
    this.fullName = function() {
        return this.first + ' ' + this.last;
    };
}
var s = new Person('Simon', 'Willison');
